⚙ CONFIGURACIÓN SUPABASE

Para usar este sistema, primero ejecutá el archivo supabase-schema.sql en tu proyecto Supabase, luego completá tus credenciales en el código fuente de este archivo.

Buscá estas líneas al inicio del JS y reemplazá los valores:

const SUPABASE_URL = 'https://xxxxxx.supabase.co'; const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIs...';

Encontrás estas credenciales en: Supabase Dashboard → Settings → API

Una vez configurado, recargá la página. Las contraseñas iniciales están en el archivo SQL.
